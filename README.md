# ido-ayx-resource

本仓库用于归档和发布多个独立的 HTML 页面。这些页面不针对特定域名或网站，而是作为通用资源存放。

## 项目简介

ido-ayx-resource 是一个静态 HTML 页面集合仓库。每个页面均为独立文件，可直接在浏览器中打开或部署到任意静态托管服务中。适合存放工具页、信息页、展示页等无需后端支持的网页资源。

## 目录说明

仓库根目录下包含多个 HTML 文件，每个文件对应一个独立页面。目录结构如下：

```
ido-ayx-resource/
├── index.html          # 仓库主页（可选）
├── page-example.html   # 示例页面
├── tool-page.html      # 工具类页面
└── info-page.html      # 信息展示页面
```

实际文件以仓库中的内容为准。可自行添加或修改文件。

## 页面归档说明

每个 HTML 页面均为独立、自包含的静态文件，不依赖外部资源（除非在文件中内联引用）。页面内容可能包含：

- 信息展示
- 简单工具
- 导航或索引
- 其他静态内容

所有页面均以 `.html` 扩展名保存，编码为 UTF-8。

## 使用方式

- 可直接在浏览器中打开 HTML 文件
- 可部署到任意支持静态文件的托管平台
- 可 fork 或 clone 仓库后自行修改
